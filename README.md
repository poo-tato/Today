# 🍎 Today (투데이)
> **"매일의 기록이 모여 더 나은 일상을 만듭니다."** > 사용자의 직관적인 지출 관리와 일상의 기록을 돕기 위해 개발된 iOS 기반 장부 앱입니다.

---

## 📝 Project Overview
* **이름**: Today
* **개발 기간**: 2026.01 ~ 현재
* **목표**: 복잡한 기능을 배제하고, '본질'에 충실한 데이터 기록 시스템 구축
* **개발자**: **Potato (Jun-seong)**

## 🚀 Key Features
준성님이 직접 설계하고 구현한 핵심 기능들입니다:
* **Data Management (CRUD)**: 일일 지출 및 수입 내역의 생성, 조회, 수정, 삭제 기능을 안정적으로 구현
* **Local Data Persistence**: 앱 재시작 시에도 데이터가 유실되지 않도록 로컬 데이터 저장 로직 설계
* **Optimized UI/UX**: 사용자 입력 동선을 최소화한 직관적인 인터페이스 및 리스트 뷰 커스터마이징
* **Error Handling**: 다양한 입력 시나리오에서 데이터 정합성을 유지하기 위한 예외 처리

## 🛠 Tech Stack
* **Language**: Swift
* **Platform**: iOS
* **IDE**: Xcode
* **Architecture**: MVC (기능 중심의 데이터 흐름 설계)

## 🧑‍💻 Developer's Reflection (800줄의 도전)
첫 프로젝트로서 약 800줄의 코드를 직접 작성하며 다음과 같은 성장을 거두었습니다.

* **코드 구조의 이해**: 초기 설계의 중요성을 깨달았으며, 점진적으로 가독성 높은 코드를 작성하기 위해 리팩토링을 반복했습니다.
* **실전 문제 해결**: 기능 구현 중 마주한 다양한 런타임 에러를 직접 디버깅하며 데이터 흐름 제어 능력을 키웠습니다.
* **확장성 고려**: 현재의 장부 로직을 바탕으로, 추후 '사용자 루틴 관리' 및 '푸시 알림' 기능을 접목한 갓생(God-Saeng) 프로젝트로의 확장을 준비하고 있습니다.
